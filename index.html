BayeSED3

A sophisticated Bayesian SED analysis tool for galaxies and AGNs

View on GitHub Download

Overview

BayeSED3 is a powerful tool for performing full Bayesian interpretation of spectral energy distributions (SEDs) of galaxies and AGNs. It combines sophisticated modeling capabilities with an intuitive interface, making it suitable for both basic and advanced astronomical research.

Key Features

Multi-component Analysis

Comprehensive SED synthesis and analysis capabilities

Stellar Population

Flexible stellar population synthesis modeling

Dust Modeling

Advanced dust attenuation & emission modeling

AGN Components

Sophisticated AGN component modeling

Installation

# Clone the repository
git clone https://github.com/hanyk/BayeSED3.git
cd BayeSED3

# Install dependencies
pip install -r requirements.txt

  • Windows: Supported through Windows Subsystem for Linux (WSL)
